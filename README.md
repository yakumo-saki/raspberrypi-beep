# raspberrypi-beep
beep for raspberry-pi (GPIO)

# how to setup
install WiringPi  
http://wiringpi.com/

Build & install WiringPi-Python  
https://github.com/WiringPi/WiringPi-Python

## setup example

On Raspberry Pi OS (aka Raspbian)

```
apt install wiringpi python3-pip
pip3 install -r requirements.txt
```
